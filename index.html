In a search for a word with a diaeresis, such as Sint Odiliënberg, it depends whether this ë is stored as one character or as "ë". This is actually a bug that should be fixed -- the entities should be folded into their raw character equivalents so all searches on them are equivalent. See also Wikipedia:Special characters.
